# RedCom Analytic

Simulador académico de redes LAN/WAN para análisis de tráfico y rendimiento.

## Ejecución

```bash
uv run python main.py
```

## Funciones actuales

- Configuración de red LAN.
- Configuración de red WAN.
- Simulación de envío de paquetes.
- Detección de congestión.
- Registro de eventos en logs.
- Cálculo de métricas.
- Exportación CSV.
- Análisis con Pandas y NumPy.
- Tres gráficas con Matplotlib.
- Guardado de métricas en SQLite.
- Historial de simulaciones.
- Pruebas unitarias iniciales.

## Pruebas

```bash
uv run pytest
```
